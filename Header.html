Adam's Curated Quotes

A collection of quotes from what I have read. The order is roughly based entirely on what I've been reading recently (descending order). I am adding highlights from my ebook reads so there's a few of those here too! You probably think this looks ugly, but this is my project and I can style it how I see fit. Best viewed with Victor Mono font installed.

Click a tag to show only quotes with that tag. You can also search for any text containined in a quote. Clicking search will add a URL fragment which you can then use to share those quotes with a friend. You can also share a selected tag in the same way.
